bootstrap-wysihtml Simple, beautiful wysiwyg editors

About

bootstrap-wysihtml is a javascript plugin that makes it easy to create simple, beautiful wysiwyg editors with the help of wysihtml and Twitter Bootstrap

Forked from bootstrap-wysihtml5, replaced unmaintained wysihtml5 with wysihtml.

View project on Github

Usage

$('.textarea').wysihtml();

Dependencies

  • bootstrap-wysihtml.js
  • bootstrap-wysihtml.css
  • wysihtml
  • Twitter Bootstrap
